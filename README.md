# Estoque

## Contribuir

1. Abra o PyCharm
2. Clique em _Get from VCS_
3. Adicione o link do projeto do GitHub
```
https://github.com/apgifro/ihc
```
4. Clone o projeto
5. Configure o interpretador e _venv_
6. Instale as dependências
```
pip install -r requirements/requirements.txt
```
